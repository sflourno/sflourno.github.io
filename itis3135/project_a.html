Project Influencer: A Guide to Being a Social Media Mogul


Project Overview


  • This project is about how to grow your current social media presence or how to start one from the beginning.
  • The intended user of this project is my boyfriend, who wants to create a social media platform to start a sneaker page.
    It can also be for anyone else who wants to become an influencer.
  • This website will have five pages. The first page will be a home/welcome page. Another page will be a sample instagram profile.
    Then tips and tricks, graphs and charts, and a page for how to go live and connect with users and businesses.
